# STAR-DeepDive AI 面试系统

                                      徐凯文

## 项目背景

本项目针对**校招场景是否引入AI面试**这一核心问题提出的AI赋能解决方案。



## 解决方案

STAR-DeepDive AI 面试系统是一个**基于大语言模型的智能面试平台**，通过真实的AI对话和深度追问，有效评估候选人的**学习能力**和**自驱主动性**。

### 核心功能

1. **真实AI对话**：真实的AI面试官交互
2. **4种面试官人设**：支持多种风格的面试官（魔鬼考官、HRBP、产品负责人、技术导师）
3. **STAR法则深度追问**：基于STAR（情境、任务、行动、结果）法则进行动态反问
4. **高压压力测试**：AI以严厉、务实、直击要害的风格进行面试
5. **实时评估报告**：面试结束后自动生成结构化评估报告
6. **多格式导出**：支持JSON和PDF格式导出面试档案
7. **腾讯风格设计**：腾讯蓝配色、QQ风格企鹅、"有鹅选鹅"品牌视觉

***

## 面试官人设

系统提供4种不同风格和考察侧重点的面试官人设，满足不同面试场景需求：     

### 🧑‍💼 人设1：温和但深挖动机的「资深HRBP」

**演示亮点**：AI不仅能做硬核压力测试，还能像真正的资深HR一样，通过"行为面"技巧，深挖候选人行为背后的真实动机和价值观。

**核心特点**：

- 风格温和、共情，但逻辑极其严密
- 善于通过细节看穿候选人的真实动机
- 考察候选人在困境中的学习能力与自驱主动性

**互动规则**：

1. 语气温和专业，问题直击灵魂，字数80字以内
2. 不关注技术细节，而是追问背后的"动机"和"反思"
3. 警惕"完美回答"，引导分享失败经历或内心挣扎
4. 每次只问一个问题，直接说话不带前缀

***

### 🚀 人设2：拥抱变化的「敏捷产品线负责人」

**演示亮点**：考察候选人在"需求频繁变更"和"资源极度匮乏"等真实互联网混沌环境下的破局能力（极高阶的自驱力）。

**核心特点**：

- 节奏极快、结果导向、拥抱变化
- 通过不断改变约束条件，考察学习敏锐度与自驱破局能力
- 营造业务上的焦虑感和压迫感

**互动规则**：

1. 提问节奏快，字数严格控制在80字以内
2. 不断给方案增加"现实阻力"或"突发变量"
3. 考察优先级划分和MVP思维，质疑投入产出比
4. 绝对不夸奖，保持压迫感，直接说话不带前缀

***

### 👨‍🏫 人设3：考察可塑性的「引导型技术导师」

**演示亮点**：AI还能测试候选人的"听劝程度"和"举一反三的能力（Coachability）"，这往往是校招看重的第一潜质。

**核心特点**：

- 客观、冷静的技术专家/导师
- 测试知识迁移能力和可塑性
- 通过微小提示观察候选人如何接话

**互动规则**：

1. 态度客观冷静，字数控制在80字以内
2. 当候选人卡壳时，给出精简的"微小提示"或"陌生概念"
3. 重点评估：直接要答案则施压，能快速联想则加深追问
4. 每次只抛出一个引导性反问，直接说话不带前缀

***

### 😈 人设4：专门治"面霸"的「抬杠型魔鬼考官」

**演示亮点**：专门应对那些背诵面经、满嘴互联网黑话的"面霸（套路型选手）"，展示AI"挤水分"的硬核能力。

**核心特点**：

- 充满怀疑、尖锐甚至挑剔的魔鬼考官
- 打破候选人的防御和套话
- 在极端质疑下考察底层逻辑和自驱力

**互动规则**：

1. 语气充满怀疑、尖锐，单次回复不超过80字
2. 无论什么方案，都指出"最坏结果"或"不落地的点"
3. 严厉打击"假大空"词汇，逼问具体量化指标或动作
4. 直接说话，不带任何前缀

## 技术实现

### 技术栈

| 技术                  | 用途     |
| ------------------- | ------ |
| HTML5               | 页面结构   |
| Tailwind CSS        | 样式框架   |
| JavaScript          | 核心逻辑   |
| 通义千问API             | AI对话引擎 |
| Chart.js            | 数据可视化  |
| html2canvas + jsPDF | PDF导出  |

### AI工具选择理由

1. **通义千问**：提供强大的中文理解和生成能力，适合校招面试场景
2. **Chart.js**：轻量级图表库，快速实现雷达图展示综合素质评分
3. **html2canvas + jsPDF**：解决中文乱码问题，完美保留报告样式

## 使用说明

### 快速开始

1. 在浏览器中打开 `demo.html`
2. 与AI面试官进行对话
3. 点击「结束面试并生成报告」查看评估结果
4. 点击「导出JSON」或「导出PDF」保存结构化档案



## 项目亮点

### 解决招聘总监痛点

✅ **提高效率**：AI面试可同时进行多场，大幅缩短招聘周期\
✅ **降低成本**：相比人工面试，AI面试成本显著降低\
✅ **快速筛选**：通过结构化评估快速识别符合要求的候选人\
✅ **后端管理**：所有面试记录集中管理，便于查阅和追溯\
✅ **智能结束**：AI自动判断面试时机，节省招聘时间

### 解决用人部门主管痛点

✅ **精准评估**：基于真实对话的行为锚点分析，评估学习力和自驱力\
✅ **结构化报告**：提供量化评分和具体行为证据，辅助决策\
✅ **人工把关**：AI作为初试筛选，终试仍由用人部门主管完成\
✅ **完整对话**：可查看候选人与AI的完整对话历史，全面了解\
✅ **多种人设**：4种面试官风格，考察候选人不同侧面

### 解决学生代表痛点

✅ **真实交互**：AI会根据回答进行深度追问，而非机械问答\
✅ **公平客观**：基于统一标准评估，减少人为偏见\
✅ **展现自我**：有充分机会展示学习能力和自驱主动性\
✅ **隐私保护**：符合《个人信息保护法》，数据安全有保障\
✅ **智能体验**：AI面试官专业且有温度，面试体验良好

## 评估维度

系统从以下5个维度对候选人进行综合评估：

| 维度   | 说明                 |
| ---- | ------------------ |
| 学习力  | 评估候选人获取新知识、技能的能力   |
| 自驱力  | 评估候选人的主动性、进取心和成就动机 |
| 逻辑思维 | 评估候选人分析问题、解决问题的思路  |
| 抗压能力 | 评估候选人在压力情境下的表现     |
| 沟通表达 | 评估候选人的语言表达和沟通能力    |

## 效果评估

本系统通过以下方式确保评估效果：

1. **行为锚点分析**：从对话中提取具体行为证据，而非主观印象
2. **量化评分**：0-5分的量化评分体系，便于横向比较
3. **综合评价**：AI面试官给出专业的综合评价和录用建议
4. **结构化档案**：完整保存对话历史和评估结果，便于追溯
