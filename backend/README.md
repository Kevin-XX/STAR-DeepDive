# STAR-DeepDive AI 面试系统 - 后端管理系统

## 📋 项目简介

这是 STAR-DeepDive AI 面试系统的后端管理系统，用于存储和管理面试记录、评估报告和对话历史。

## 🚀 快速开始

### 前置要求

- Node.js (推荐 v14 或更高版本)
- npm (Node.js 包管理器)

### 安装步骤

1. **进入后端目录**
```bash
cd backend
```

2. **安装依赖**
```bash
npm install
```

3. **启动服务器**
```bash
npm start
```

服务器将在 `http://localhost:3000` 启动。

## 📖 使用说明

### 1. 启动后端服务器

在 `backend` 目录下运行：
```bash
npm start
```

看到以下输出表示启动成功：
```
========================================
  STAR-DeepDive AI 后端服务器
  有鹅选鹅 · 智能面试新体验
========================================
服务器运行在: http://localhost:3000
管理页面: http://localhost:3000
数据文件: .../backend/interviews.json
========================================
```

### 2. 使用前端保存面试记录

1. 在前端完成面试后，点击"💾 保存到后端"按钮
2. 确保后端服务器正在运行
3. 保存成功后会收到提示

### 3. 查看管理后台

访问 `http://localhost:3000` 即可查看管理后台。

管理后台功能：
- 📋 查看所有面试记录列表
- 📊 查看详细的能力评分和评估报告
- 💬 查看完整的对话历史
- 🗑️ 删除不需要的面试记录

## 🔌 API 接口

### 保存面试记录

```
POST /api/interview
Content-Type: application/json

{
  "report": { ... },        // 评估报告数据
  "conversation": [ ... ],  // 对话历史
  "persona": "devil"        // 面试官人设
}
```

### 获取所有面试记录

```
GET /api/interviews
```

### 获取单条面试记录详情

```
GET /api/interview/:id
```

### 删除面试记录

```
DELETE /api/interview/:id
```

## 🎨 管理后台特色

- **腾讯风格设计**：使用腾讯蓝渐变和QQ风格企鹅
- **直观的评分展示**：5项能力评分一目了然
- **完整对话历史**：可以查看完整的面试对话
- **移动端友好**：响应式设计，支持各种设备
- **操作简单**：点击即可查看详情，一键删除

## 📝 注意事项

1. **数据存储**：所有数据保存在 `interviews.json` 文件中，请妥善保管
2. **备份建议**：定期备份 `interviews.json` 文件
3. **端口占用**：确保 3000 端口未被占用，或修改 `server.js` 中的 `PORT` 变量

## 🛠️ 技术栈

- **后端框架**：Express.js
- **数据存储**：JSON 文件（简单易用）
- **前端UI**：Tailwind CSS
- **跨域支持**：CORS

## 🔒 隐私保护

根据《隐私保护承诺》：

- 面试数据仅在本地管理后台可见
