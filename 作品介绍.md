# 作品介绍：STAR-DeepDive AI 面试系统

## 项目背景

针对校招场景中传统面试效率低下、难以快速筛选具有学习能力和自驱主动性候选人的痛点，我独立设计并开发了这款基于大语言模型的智能面试平台。项目源自腾讯AI-HR STAR笔试作业，通过AI赋能解决实际招聘问题。

## 技术实现

### 前端技术栈
- **HTML5 + Tailwind CSS**：构建现代化、腾讯风格的用户界面
- **JavaScript**：实现核心业务逻辑和交互
- **通义千问API**：接入大语言模型实现AI面试官对话
- **Chart.js**：实现5维能力评分雷达图可视化
- **html2canvas + jsPDF**：解决中文乱码问题，完美导出PDF报告

### 后端技术栈
- **Node.js + Express.js**：轻量级后端服务器框架
- **RESTful API**：设计标准化接口，支持前后端分离
- **CORS跨域支持**：确保前端正常调用后端接口
- **JSON文件存储**：简单高效的数据持久化方案

## 核心功能

1. **4种面试官人设**：支持魔鬼考官、HRBP、产品负责人、技术导师四种风格，考察候选人不同侧面
2. **真实AI对话**：基于STAR法则进行深度追问，打破套话
3. **结构化评估报告**：学习力、自驱力、逻辑思维、抗压能力、沟通表达5维量化评分
4. **后端管理系统**：集中保存和查看所有面试记录，支持删除操作
5. **智能结束面试**：AI自动判断5-6轮表现好则结束，敷衍者先警告后结束
6. **自动保存**：报告生成后自动保存到后端，无需手动操作
7. **隐私保护**：符合《个人信息保护法》，保障候选人数据安全
